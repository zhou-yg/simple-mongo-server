# simple-mongo-server

# use case

> [localhost]:[port]/[dbName]/[collectionName]/[method]/[key]/[value]

```
{
  [key]: value
}
```

> [localhost]:[port]/[dbName]/[collectionName]/[method]/[key] ? arg=[JSON_STRING]




```
{
  [key]: arg
}
```

[localhost]:[port]/[dbName]/[collectionName]/[method] ? arg=[JSON_STRING]

```
{
  arg: arg
}
```
